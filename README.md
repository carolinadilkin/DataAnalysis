# Análise de Dados de Salários de Profissionais de Dados

## Visão Geral
Este projeto realiza uma análise exploratória (EDA) e visualização de dados de salários de profissionais de dados. O objetivo principal é identificar tendências salariais, distribuições e os fatores que as influenciam, como nível de experiência, localização e regime de trabalho (remoto).

## O que o Projeto Faz
- **Limpeza de Dados:** Trata valores ausentes, remove linhas com dados incompletos e converte tipos de dados para garantir a integridade da análise.
- **Análise Exploratória:** Examina a distribuição de níveis de experiência, salários e a proporção de trabalho remoto.
- **Visualização de Dados:** Utiliza gráficos de barras, histogramas, boxplots e mapas coropléticos para ilustrar padrões e insights sobre os salários.
- **Análise Específica:** Foca na média salarial para Cientistas de Dados por país e nível de experiência.

## Tecnologias Utilizadas
- **Python:** Linguagem de programação principal.
- **Pandas:** Para manipulação e análise de dados.
- **Matplotlib e Seaborn:** Para criação de gráficos estatísticos e visualizações.
- **Plotly Express:** Para gráficos interativos, incluindo mapas coropléticos.
- **Pycountry:** Para conversão de códigos de países (ISO 2 para ISO 3).

## Objetivo
O principal objetivo é fornecer insights claros sobre o mercado de trabalho para profissionais de dados, ajudando tanto empregadores quanto profissionais a entenderem as expectativas salariais e os fatores que as moldam em diferentes contextos geográficos e de experiência.

## Status do Projeto
**Concluído:** A análise e as visualizações foram realizadas e documentadas neste notebook.
